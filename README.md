# USB Role Switch

KernelSU WebUI module for switching USB Type-C power and data roles independently on rooted devices.

Author: GitHub ciallothu
Repository: github.com/ciallothu/usb_role_switch

Commands:

- `power-source`
- `power-sink`
- `power-toggle`
- `data-host`
- `data-device`
- `data-toggle`
- `source-host`
- `sink-device`
- `pair-toggle`
- `status`
- `log`
- `clear-log`
- `diag`

The module does not force USB roles at boot.
